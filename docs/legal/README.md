# Documentación legal

Esta carpeta contiene los textos legales del sistema, principalmente el consentimiento informado para tratamiento de datos biométricos según la Ley 25.326 y la Resolución AAIP 255/2022.

## Archivos previstos

- `consentimiento_v1.md`: texto del consentimiento que el docente firma al registrar su modelo facial. Versionado: cuando cambie, se crea `consentimiento_v2.md` y se mantiene el anterior.
- `politica-privacidad.md`: política general de tratamiento de datos personales.
- `derechos-arco.md`: procedimiento para que el docente ejerza derechos de Acceso, Rectificación, Cancelación y Oposición.

Los documentos se redactarán durante el Sprint 3 (Docentes + Consentimiento).
